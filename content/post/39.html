+++ title = '未来' date = 2025-01-22 draft = false description = '' categories = ['故事'] +++ 一个平常的日子中，一所普通的医院里传来一阵婴儿的啼哭声。
你出生了。
家里的经济不算良好。
父母打了一辈子工，一直想找个能实现他们愿望的人，
而你就是那个他们期望的人。
刚出生的你就已经背负了父母给你的使命：将来出人头地，别像普通人一样劳苦一辈子。
虽然你那时连眼都没有睁开。

慢慢的你长大了。
和千千万万的孩子一样度过了童年、少年时光。
但总觉得自己并非那千千万万中的一员。
或许是在羊水中母亲通过脐带传递了他们的思想。
“我是特别的”，你心中暗自窃喜。
我和他们可不一样。

高中时你的成绩并不算理想，总被父亲管教着学习。
不过还好，你找到了自己真正想做的事情，并决定报考关于这门的专业。
在历经了一些曲折后总算考上了如愿的大学。
晚上父亲对你说：你要好好学习，将来出人头地，我和你妈把希望都寄托在你身上了。
你看着他棕黑的脸颊和开裂的手指，你点了点头，想着一定不会辜负他们的期望。

到了开学的日子。
和拎着行李父亲一起站在大门的那一刻，你抬头看到了高耸的教学楼。
“终于”。你长舒一口气。
这使你充满了决心，想象着同学，老师和教学内容，决定要好好学习，为了你操劳的父母。
继而幻想到毕业后的工作，赚到钱回家后父母的笑脸。
光想到这些嘴角已经止不住的上扬了。
但你又看到了千千万万的学生。
他们和你一样。
你的笑容有些僵硬了。
人的海洋将你推进大门。
“或许我并没有那么特别”。

几周后你慢慢认识了你的舍友。
开始时你们一起吃饭，一起上课。
渐渐地有人不愿去上课了。
规劝了两句后依然无果，你便走了。
后来他们让你帮忙签到，你答应了。
有时叫你去网吧，但你总是拒绝。
“学习有啥用啊？毕业了还不是打工。”
他说罢便躺到了床上打开了游戏。
但你有自己的目标。
他们外出回来给你带些吃的，还说谢谢你帮他签到。
你谢了他们，但心想这是不对的。
但也不再劝说。

一次聊天问到了大家的家庭，你在床上躺着听到。
他说他家有个公司。
来上学不过混个文凭，反正毕业后就去他爸的公司了。
他们的对话让你想到了你的父母。
普通的劳动工人。
继而让你想到他们光鲜亮丽的衣裳和大手大脚的开销。
转头又想到自己平时紧凑的生活费。
但你心里有股劲，觉得靠自己的努力肯定也能翻身。
靠父母、有靠山算什么。
但你自己知道，内心还是有些羡慕和嫉妒的味道。
你沉默不语。
“我是特殊的吗？成绩普通，样貌普通，家庭甚至算不上普通”
“大概我也只是一个普通人”。

毕业典礼那天你们站在一起，微笑着看向镜头。
一，二，三，“茄子！”
大家将手中的毕业帽抛向天空。
你抬头看向了漫天的毕业帽。
黑压压一片盖住了本该照向你的阳光。
帽子落下，短暂的兴奋后。
叽哩哇啦的嘈杂充满了操场。

你缓缓看向了镜头。
你期待着未来。
你畅想着未来。
你憧憬着未来。
你恐惧着未来。
