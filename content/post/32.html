+++ title = '你能听到我说话吗？' date = 2024-10-10 draft = false description = '' categories = ['感想'] tags = ['诗歌'] +++ 从万花镜后看，我从不同角度看到了无数个你，那个才是真正的呢？
— 把那玩意扔了吧。

不再关心新闻事件，不再关心房子汽车，不再关心粮食水果。
不关心别人，也不关心自己。
不要刺激，不要紧张，不要兴奋，不要空虚，都别来破坏宁静，一粒尘埃都能激起一片涟漪的宁静，保持现状就好。

在荒野草地中的床上，钻到被窝里，耳边只有微风吹拂杂草的沙沙声。
但这次一反往日常态，即没有星辰，也没有夕阳，只有不算通透的空气。
好安静啊，你能听到我说话吗？你能听到我说话吗？ 你能听到我说话吗？ 你能听到我说话吗？……
…………
真的好安静。
